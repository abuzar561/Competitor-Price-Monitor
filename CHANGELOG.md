# Changelog

All notable changes to this project will be documented here.

## 1.0.0 - 2026-05-02

- Published the project as a clean competitor price monitoring automation.
- Added a Selenium-based Python scraper with configurable product URL, selector, target price, currency, and n8n webhook.
- Added a sanitized n8n workflow template for Google Sheets logging and Gmail alerts.
- Added setup, workflow, payload, troubleshooting, contribution, security, and validation documentation.
- Added GitHub Actions validation for repository quality checks and Python compilation.
